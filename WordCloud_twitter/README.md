# Twitter WordCloud
Get a Word Cloud for any user on Twitter
## Acquire the Tweets
run as follows
```
python3 scrape_twitter.py > out1.txt

```
and then, enter the username of any twitter account

## Clean the tweets
run as follows
```
python3 clean.py > fin.txt

```
# Make the WordCloud
run as follows
```
python3 word_cloud.py

```
